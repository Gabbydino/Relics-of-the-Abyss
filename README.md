# ⚔ Relics of the Abyss

Um roguelike de exploração de masmoras procedural, jogável direto no navegador — sem instalação, sem dependências externas além de fontes do Google Fonts.

🔗 Repositório: [github.com/Gabbydino/Relics-of-the-Abyss](https://github.com/Gabbydino/Relics-of-the-Abyss)

## Como jogar

Abra `index.html` no navegador (ou acesse via GitHub Pages).

- **Mover:** WASD ou setas
- **Atacar:** ande contra o inimigo
- **Habilidade especial:** ESPAÇO
- **Pausa:** ESC

## O que tem no jogo

- Geração procedural de masmorra a cada andar (9 andares, 3 biomas: Cavernas → Ruínas Antigas → Catacumbas)
- 4 classes jogáveis (Guerreiro, Arqueiro, Mago, Paladino), com desbloqueio progressivo por recorde de andar
- Fog of war com linha de visão real
- Iluminação dinâmica de tocha, texturas de piso proceduais e vinheta atmosférica
- Inventário + 8 slots de equipamento, 6 raridades (com efeitos únicos em itens lendários/míticos)
- Salas especiais: Loja, Fonte de Cura, Sala do Tesouro
- 12 monstros com IAs distintas + 3 chefes com ataques especiais
- Eventos aleatórios, baús (alguns trancados), sistema de XP/nível
- Save automático via LocalStorage (recorde de andar, desbloqueios de classe)
